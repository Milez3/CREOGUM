CREOGUM prototype reference
=============================

The April 2026 Developer Brief states that the single-file prototype HTML is the
source of truth for approved copy and motion.

1. Save your prototype file here as:  prototype.html
2. Diff against content/site.ts and components under components/creogum/
3. Prefer updating content/site.ts for text so the Next.js app stays maintainable.

If prototype.html is missing, the app uses content/site.ts (aligned to the PDF brief).
